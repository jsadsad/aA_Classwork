# Embedded Ruby (ERB)

- HTML templates augmented with Ruby Code
- Non-returning tags are: `<% Ruby code here %>` to execute Ruby code WITHOUT a return value.
- Returning tags: `<%= Ruby code here %>` executes Ruby code WITH a return value.
